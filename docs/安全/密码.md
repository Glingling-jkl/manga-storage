# 密码

## 密码设置规则
注册时需设置**登录密码**，规则如下：
- 长度至少 **6 位**（推荐使用 12 位以上的复杂密码）
- 建议包含大小写字母、数字和特殊符号（如 `!@#$`），以提高安全性

## 如何修改密码
1. 在主页右下角点击 **三条杠按钮**（菜单）。
2. 在展开的子菜单中，点击 **小设置**。
3. 进入设置页面后，点击 **修改密码**。
4. 输入当前密码，再输入新密码并确认。
5. 保存后退出登录，下次登录时需使用新密码。

## 忘记密码怎么办？
目前本网站**暂不支持自助找回密码**。如忘记密码，请联系管理员进行重置。管理员会在核实身份后协助你设置新密码。

## 密码安全建议
- 不要将密码告知他人，包括管理员（管理员不会索要密码）。
- 避免使用生日、姓名等易猜测的信息作为密码。
- 不要在多个网站重复使用同一密码。
- 如果怀疑密码泄露，立即修改密码并检查账户近期活动。
