# 第二层密码

## 什么是第二层密码？
第二层密码是你在注册时**强制设置**的额外安全密码，用于增强账户保护。当系统检测到登录设备或 IP 地址与以往不同时，会要求你输入第二层密码进行二次验证，确认是本人操作。

## 第二层密码规则
- **必须是 8 位数字**（不支持字母或符号）
- 注册时强制设置，不可留空

## 第二层密码与登录密码的区别
| 项目 | 登录密码 | 第二层密码 |
|------|---------|-----------|
| 用途 | 日常登录验证 | 异常设备/IP 时的二次验证 |
| 长度要求 | 至少 6 位（推荐 12 位以上） | 固定 8 位数字 |
| 修改方式 | 个人设置中修改 | 个人设置中修改 |
| 是否强制 | 是 | 是 |
| 是否可重置 | 可联系管理员重置 | 可联系管理员重置 |

## 如何修改第二层密码
1. 在主页右下角点击 **三条杠按钮**（菜单）。
2. 在展开的子菜单中，点击 **小设置**。
3. 进入设置页面后，点击 **修改二密**。
4. 输入当前第二层密码（若未设置过则留空），再输入新的 8 位数字密码并确认。
5. 保存后立即生效，下次触发设备校验时将使用新密码。

## 忘记第二层密码怎么办？
如果忘记了第二层密码，且因设备变更被锁在账户外：
- 请联系管理员，提供能证明身份的信息（如注册时的姓名、常用 IP 等）。
- 管理员核实后，会为你重置第二层密码，重置后请尽快登录并修改为自己记得的 8 位数字。

## 安全提示
- 第二层密码应与登录密码不同，且不要使用连续数字（如 12345678）或重复数字（如 88888888）。
- 请勿将第二层密码记录在易泄露的地方（如便签、未加密的云笔记）。
- 如怀疑第二层密码泄露，立即修改并检查账户近期是否有异常登录记录。
