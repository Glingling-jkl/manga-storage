# 安全协议

本网站（manga-site-002）致力于为用户提供安全、可靠的服务。我们采取以下措施保护你的账户和数据安全。

## 账户安全
- **强制第二层密码**：每个账户注册时必须设置第二层密码，用于设备变更时的二次验证，防止未授权登录。
- **IP 校验机制**：系统会记录常用登录 IP，当检测到新设备或 IP 变化时，要求输入第二层密码进行身份确认。
- **权限隔离**：普通用户、管理员、system 组权限严格分离，普通用户无法访问管理功能，管理员也无法查看普通用户的密码等敏感信息。

## 数据传输与存储
- 本网站所有通信均通过 **Cloudflare** 提供的 SSL/TLS 加密，确保数据在传输过程中不被窃听或篡改。
- 用户密码（包括登录密码和第二层密码）在存储前经过哈希加密，即使数据库泄露也无法还原原始密码。
- 上传的文件（如图片、评论附件）存储在 **GitHub** 仓库中，访问受限于公开链接，但建议不要上传包含个人敏感信息的文件。

## 漏洞与响应
- 如发现安全漏洞，请通过“Bug”页面或指定渠道及时反馈，我们会尽快修复并发布更新说明。
- system 账户仅用于测试网站基础功能与安全性，不会用于日常操作或查看用户隐私。

## 用户责任
- 请妥善保管你的密码和第二层密码，不要分享给他人。
- 建议定期修改密码，避免使用与其他网站相同的密码。
- 如发现账户异常（非本人操作），立即联系管理员修改密码并检查近期活动。

本安全协议将随网站功能更新适时调整，如有重大变更将另行通知。
