# 2026年3月13日更新

**更新时间：2026年3月13日 至 2026年3月15日**

## 更新内容

### 1. 增加权限管理系统
- 引入三个用户组：**普通用户组**、**管理员组**、**system组**。
- 普通用户组：仅可浏览漫画、发表评论。
- 管理员组：除普通用户权限外，可上传漫画、删除自己上传的漫画、**删除普通用户**。
- system组：系统保留账户，仅开发者使用，用于测试安全性与基础功能，不参与日常运营。**仅system账户可修改用户组（普通 ↔ 管理员）**。
- 权限控制细化到页面和按钮级别，不同用户组看到的界面元素不同。

### 2. 新增用户管理页面
- 管理员可访问“用户管理”页面，查看所有注册用户列表。
- 支持按用户名、注册时间排序。
- 支持**管理员删除普通用户**（删除后用户账户及关联数据将被移除）。

### 3. 新增漫画上传与管理功能（仅管理员）
- 管理员可访问“上传漫画”页面，填写漫画名称、封面、章节等信息。**支持上传单个漫画**（每次仅能上传一个漫画，包含其封面和章节图片）。
- 管理员可在**“管理漫画”**中查看自己上传的漫画，**仅支持删除操作**（不支持编辑）。
- 删除漫画时，同时删除关联的图片文件（从 GitHub 存储中移除）。
