# 2026年3月27日更新

**更新时间：2026年3月27日 至 2026年3月29日**

## 更新内容

### 1. 优化并美化上传漫画页面
- 重新设计上传漫画页面，采用卡片式布局，更直观。
- 支持拖拽上传图片，提升操作效率。
- 增加上传进度条，实时显示上传状态。
- 增加图片预览功能，上传前可查看缩略图。

### 2. 评论区增加至第二层（嵌套回复）
- 支持对已有评论进行**回复**，形成二级嵌套评论。
- 回复会自动缩进，清晰展示对话层级。
- 新增评论点赞功能（可选）。

### 3. 手机端主页漫画由一列变为两列
- 针对移动端（宽度 < 768px）优化漫画列表布局，从单列改为双列网格。
- 每个漫画卡片显示封面、标题，点击跳转详情页。
- 调整卡片间距和字体大小，更适应手机触控操作。

### 4. 优化并美化漫画详情页
- 重新设计漫画详情页头部，显示封面、作者、简介等。
- 修复了部分设备上阅读页面左右滑动时误触菜单的问题。

### 5. 文化传输逻辑优化，网站速度更快
- 优化图片加载策略：从 GitHub 获取图片时使用 Cloudflare 缓存，减少回源请求。
- 对静态资源（CSS、JS）进行压缩和合并，减少请求次数。
- 启用浏览器缓存策略，二次访问速度明显提升。
- 修复了部分网络环境下因 GitHub 连通性导致图片加载失败的 bug，增加重试机制。
