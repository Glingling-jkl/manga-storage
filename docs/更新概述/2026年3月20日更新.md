# 2026年3月20日更新

**更新时间：2026年3月20日 至 2026年3月22日**

## 更新内容

### 1. 增加评论区功能并修复相关问题
- 新增评论区功能，支持用户对漫画章节进行评论。
- 评论支持 Markdown 简单格式（粗体、斜体、链接）。
- 修复了评论提交后页面未自动刷新的问题。
- 修复了评论时间显示错误（时区问题）的 bug。
- 增加评论删除规则：用户仅可删除自己**发布后 5 分钟内**的评论，超过 5 分钟不可删除（防止恶意删评）。

### 2. 修复用户管理页面的显示 Bug
- 修复了用户管理页面在某些分辨率下表格错位的问题。
- 修复了管理员在修改用户组后，页面状态未及时刷新的问题。
- 修复了重置密码时未清空旧 session 的潜在安全漏洞。
