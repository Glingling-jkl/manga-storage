# 隐私协议

**更新日期：2026年3月27日**  
**生效日期：2026年3月27日**

manga-site-002（以下简称“本网站”）由个人运营，主要面向校内朋友提供漫画浏览、评论及管理服务。本隐私协议旨在清晰说明本网站如何收集、使用、存储和保护你的个人信息，以及不同用户组在数据处理上的差异。

## 1. 信息收集
在使用本网站的过程中，我们可能收集以下类别的信息：

### 1.1 你主动提供的信息
- **注册与身份信息**：你在注册时填写的**姓名**、自行设置的**登录密码**及**第二层密码**（用于设备校验失败时的二次验证）。
- **评论与上传内容**：你在漫画评论区发表的**评论内容**、以及你自愿**上传的文件**（如图片、头像等）。

### 1.2 自动收集的信息
- **技术信息**：当你访问本网站时，系统会自动记录你的**IP地址**、**访问时间**、**浏览器类型**以及通过Cloudflare获取的基础访问日志。
- **行为记录**：系统会自动记录**评论发布时间**、**漫画浏览记录**（仅用于功能正常展示，不作跨用户分析）。

## 2. 信息使用目的
我们收集的信息将用于以下特定目的：

| 使用目的 | 说明 |
|---------|------|
| **安全校验** | IP地址用于判断当前设备环境是否可信。若校验不通过，系统将要求你输入**第二层密码**进行身份验证，以保护账户安全。 |
| **核心功能运行** | 包括漫画浏览、评论互动、上传漫画（仅管理员）、用户组权限管理等。 |
| **权限控制** | 根据你所在的用户组（普通用户、管理员、system）提供对应的功能权限，并限制越权访问。 |
| **维护与优化** | 分析访问模式以排查故障、优化性能。此环节仅使用聚合数据，不单独识别个人身份。 |

## 3. 用户组与数据访问权限
本网站设有三个用户组，不同组别对数据和功能拥有不同权限：

- **普通用户组**  
  可以查看漫画、发表评论。普通用户只能查看和修改**自己的**个人信息（姓名、密码、第二层密码），无法查看或管理其他用户的数据。

- **管理员组**  
  除普通用户的所有权限外，还可以**上传漫画**、**删除自己上传的漫画**、**管理普通用户组**（如调整普通用户状态）。管理员在履行管理职责时，可能会接触到普通用户的公开信息（如评论内容），但不会访问普通用户的密码等敏感凭证。

- **system 用户组**  
  该组仅包含**一个固定账户**，由本网站开发者使用，**不对外开放**。该账户具有最高权限，但**仅用于测试网站基础功能与安全性**，不会用于日常运营或查看普通用户的隐私内容。开发者承诺不会通过该账户获取或泄露用户个人信息。

## 4. 第三方服务
本网站使用了以下第三方服务，这些服务可能在各自隐私政策下处理你的部分信息：

- **Cloudflare**：提供网站托管、CDN加速、安全防护及基础访问日志记录。  
  *隐私政策：https://www.cloudflare.com/zh-cn/privacypolicy/*

- **GitHub**：用于代码存储及用户上传的内容（如漫画图片、用户上传文件）的托管。  
  *隐私政策：https://docs.github.com/cn/site-policy/privacy-policies/github-privacy-statement*

上述第三方可能将数据存储于其全球服务器中，使用本网站即视为你同意此类处理。

## 5. 信息存储与跨境传输
你的个人信息及上传内容将存储在**Cloudflare**与**GitHub**的服务器上（可能位于美国或其他国家/地区）。由于本网站使用国际服务提供商，你的信息可能被传输至你所在国家/地区以外的司法管辖区。此类传输将遵循相关服务商的隐私保护机制。

## 6. 你的权利
你可以通过登录账户随时**修改你的登录密码**及**第二层密码**。  
你可以**查看和删除**自己发表的评论及上传的文件。

**请注意**：  
- 本网站**暂不支持账号注销功能**。如果你希望停止使用，可以停止登录并自行清除本地记录。  
- 对于权限不足的内容页面（例如普通用户尝试访问管理功能），系统将拒绝访问并予以提示。  
- 如你对个人信息处理有任何疑问或需要协助，可通过网站内公告的联系方式与我们沟通。

## 7. 儿童保护
本网站不面向**14周岁以下**儿童，也不主动收集儿童个人信息。如你未满14周岁，请勿注册或使用本网站。如果我们发现无意中收集了儿童个人信息，将尽快予以删除。

## 8. 协议更新
我们可能会适时更新本隐私协议，以反映功能变化或法律法规要求。更新后的协议将在本页面公布，并修改“更新日期”。若涉及重大变更（如数据使用目的改变），我们将在网站内发布通知。继续使用本网站即表示你同意修订后的协议。
