# 反馈 Bug

感谢你帮助改进本网站！如果你在使用过程中遇到异常（功能无法使用、页面显示错误、安全漏洞等），请按照以下格式向我们反馈。

## 如何反馈？
你可以通过网站内提供的联系方式（如隐私中的隐私协议）提交反馈，为了提高处理效率，请尽量包含以下信息：

### 反馈模板
- **问题类型**：功能异常 / 页面显示错误 / 权限错误 / 安全漏洞 / 其他
- **发生时间**：YYYY-MM-DD HH:MM
- **你的用户组**：普通用户 / 管理员
- **操作步骤**：详细描述你在出现 Bug 前做了什么（例如：登录后点击“上传漫画”，然后……）
- **实际结果**：出现了什么错误？（附上截图或文字描述）
- **预期结果**：你希望正确情况下应该发生什么？
- **浏览器及版本**：Chrome / Firefox / Safari 等，版本号（可选）
- **网络环境**：学校Wi-Fi / 家庭宽带 / 移动网络（可选）

## 示例
> **问题类型**：功能异常  
> **发生时间**：2026-03-29 14:20  
> **你的用户组**：管理员  
> **操作步骤**：登录后，点击“上传漫画”，选择图片后点击“提交”按钮  
> **实际结果**：页面一直转圈，最后提示“上传失败”  
> **预期结果**：漫画应该成功上传并显示在列表中  
> **浏览器**：Chrome 120  
> **网络环境**：学校Wi-Fi

## 我们会如何处理
- 我们会尽快核实并修复问题，修复进度将在“更新日志”中说明。
- 如果问题涉及安全漏洞，我们会在修复后及时发布安全公告。

再次感谢你的反馈！
