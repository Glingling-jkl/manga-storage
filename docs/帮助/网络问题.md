# 网络问题

## 网站无法访问或加载缓慢怎么办？
- 首先，请检查你的网络连接是否正常。
- 本网站使用 Cloudflare 提供 CDN 加速和安全防护，部分地区网络波动可能导致访问缓慢。你可以尝试刷新页面或更换网络环境（如切换 Wi-Fi / 移动数据）。
- 如果长时间无法访问，请确认是否被本地网络防火墙屏蔽（部分学校或公司网络可能限制某些端口）。你也可以通过其他设备（如手机热点）测试是否可访问。

## 为什么登录时提示“设备校验失败”？
当系统检测到你的 IP 地址与上次登录不同时，会触发安全校验。你需要输入**第二层密码**才能继续登录。这是为了保护你的账户不被他人盗用。

如果你忘记了第二层密码，请联系管理员重置。

## 图片（漫画页）加载不出来怎么办？
- 请尝试刷新页面或清除浏览器缓存（Ctrl+Shift+Delete）。
- 检查网络是否稳定，有时图片加载失败可等待几秒后重试。
- 若大量用户反馈同一漫画无法加载，可能是图片存储服务（GitHub）临时波动，我们会尽快修复。

## 关于 GitHub 存储库的国内连接问题
本网站的漫画图片及部分资源存储在 **GitHub** 上。由于 GitHub 在国内部分网络环境下可能连接不稳定，可能会导致图片加载缓慢或偶尔无法显示。遇到这种情况：
- 可以尝试切换网络（如从校园网切换至手机热点）后刷新。
- 也可稍等片刻再试，GitHub 的连通性会随时间变化。
- 如果问题持续存在，请通过 Bug 反馈渠道告知我们，我们会考虑优化存储方案。

## 提交评论时提示失败
可能是网络超时或服务器临时繁忙。建议复制评论内容后刷新页面重试。如果反复失败，请将你的操作步骤和错误提示通过“Bug”页面反馈给我们。
