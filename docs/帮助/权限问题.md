# 权限问题

## 我的账号为什么无法上传漫画？
只有**管理员**用户组才拥有上传漫画的权限。如果你在注册时被分配为**普通用户组**，则只能浏览漫画和发表评论。若你需要上传漫画，请联系管理员为你提升权限。

## 为什么我看不到某些页面或按钮？
本网站根据用户组（普通用户、管理员、system）控制功能访问：
- **普通用户**：仅可查看漫画、发表评论，无法访问管理后台。
- **管理员**：可上传漫画、删除自己上传的漫画、管理普通用户。
- **system**：系统专用账户，不对外开放。

如果你确信自己应当拥有某项权限但无法使用，请检查你是否已登录正确账号，或联系管理员核对用户组设置。

## 为什么我无法删除我自己的评论？
你可以删除自己发布的评论，但**仅限发布后5分钟以内**。超过5分钟的评论将无法撤销，这是为了保持讨论的连贯性和防止恶意删评。如果你确实需要删除较早的评论，可联系管理员协助处理。

## 什么是第二层密码？
第二层密码是你在注册时强制设置的额外安全密码。当系统检测到你的登录设备异常（例如IP地址变化）时，会要求你输入第二层密码进行验证，以保护账户安全。你可以随时在个人设置中修改该密码。
